# ZuiIT21

[中文](README.md) | [日本語](README_ja.md)

SSH（Spring、Struts、Hibernate）フレームワークを使用して開発したニュース配信システムです。

## プロジェクト紹介

これは、2017年に制作した大学の卒業制作です。

当時は仕事をしながらこのプロジェクトを作っていたので、毎日かなり忙しかった記憶があります。今振り返ると、細かいことはもうあまり覚えていません。ただ、ずっとコードを書いたり、画面を直したり、資料を調べたりして、何とか最後まで完成させたことだけはよく覚えています。（￣▽￣）

システムはB/S構成で、主にニュースを閲覧するフロント画面と、記事やユーザーを管理するバックエンド画面に分かれています。一般ユーザーはニュースを分類ごとに閲覧したり、記事を評価したりできます。登録・ログイン後は、自分のプロフィールも管理できます。管理者や投稿者は、管理画面から記事を投稿し、記事、分類、ユーザー、管理者情報などを管理できます。

今見ると、使っている技術も画面のデザインもかなり時代を感じますし、当時直しきれなかった小さな不具合も残っています。卒業制作の発表では、先生から「画面は少し古いけれど、コードはちゃんとしている」と言ってもらいました。この言葉は今でも覚えています。∑( 口 ||

## 主な機能

### フロント画面

* 分類ごとのニュース閲覧
* 記事タイトル、本文、投稿者、日時、閲覧情報の表示
* 記事の評価機能
* ユーザー登録、ログイン、ログアウト
* ログイン時の認証コード
* プロフィール情報の確認・変更

### 管理画面

* 管理者・投稿者のログイン
* リッチテキストエディタを使用した記事投稿
* 記事画像のアップロード
* 記事の検索、変更、削除
* 記事分類の追加、変更、削除
* ユーザー情報の確認・管理
* ユーザーアカウントの有効化・無効化
* 管理者アカウントの作成、変更、削除
* 管理者レベルに応じた権限制御

## ユーザー区分

システムには、以下の4種類のユーザー区分があります。

### スーパー管理者

システム全体に関する主な管理権限を持ち、以下の情報を管理できます。

* ニュース記事
* 記事分類
* 一般ユーザー
* システム管理者
* 投稿者アカウント

### システム管理者

管理画面を利用し、記事、分類、ユーザーなどを管理できます。ただし、一部の管理者管理機能は権限レベルによって制限されます。

### 投稿者

管理画面から記事を投稿し、自分が投稿した記事を確認、変更、管理できます。

### 一般ユーザー

フロント画面でニュースを閲覧・評価できます。また、ユーザー登録とログインを行い、プロフィール情報を管理できます。

## システム構成

このプロジェクトは、従来型のJava Webのレイヤー構成を採用しています。

```text
ブラウザ
  │
  ▼
JSP画面
  │
  ▼
Struts Action
  │
  ▼
Service ビジネスロジック
  │
  ▼
DAO データアクセス
  │
  ▼
Hibernate
  │
  ▼
MySQL
```

各レイヤーの主な役割は以下のとおりです。

* **JSP**
  フロント画面と管理画面の表示を担当します。

* **Struts 2**
  リクエストの受け取り、Actionの呼び出し、画面遷移を担当します。

* **Spring**
  オブジェクト管理、依存性注入、Service層とDAO層の接続を担当します。

* **Hibernate**
  Javaオブジェクトとデータベーステーブルのマッピング、およびデータ永続化を担当します。

* **MySQL**
  記事、分類、ユーザー、管理者などのデータを保存します。

## 記事投稿

管理画面の記事編集ページでは、KindEditorを使用しています。記事タイトル、分類、公開日時を設定し、本文をリッチテキスト形式で編集できます。

## データベース設計

システムでは主に以下のデータを管理しています。

| データ種別  | 主な内容                                   |
| ------ | -------------------------------------- |
| ニュース記事 | タイトル、本文、投稿者、日時、閲覧数、人気度、分類              |
| 一般ユーザー | アカウント、パスワード、ニックネーム、性別、連絡先、生年月日、アイコン、状態 |
| 管理者    | ログイン名、パスワード、名前、権限レベル、状態                |
| 記事分類   | 分類名                                    |
